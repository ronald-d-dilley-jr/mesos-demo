# worker

### Environment Variables
| Variable   | Description                                                             |
| ---------- | ----------------------------------------------------------------------- |
| DEMO_WORKER_SEED | Seed to use for random generator.  (Default - Current time) |
| DEMO_WORKER_STANDARD_RANGE | |
| DEMO_WORKER_VARIATION_RANGE | |
| DEMO_WORKER_VARIATION_PERCENTAGE | |
